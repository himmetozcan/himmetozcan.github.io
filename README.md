# A. Himmet Ozcan Portfolio

Static GitHub Pages portfolio for selected AI engineering projects.

Live URL after GitHub Pages deploys:

```text
https://himmetozcan.github.io/
```

## Updating project visuals

The current site uses CSS concept placeholders. Later, generated or sanitized project visuals can be added under:

```text
assets/project-visuals/
```

Recommended image specs:

- `1600x900` or `1200x675`
- `.webp` or optimized `.png`
- concept visuals for client work, not real confidential screenshots
- real screenshots only for public products such as SilentProof or vtry.ai

Suggested filenames:

```text
silentproof.webp
vtry-ai.webp
pamukkale-onprem-ai.webp
utility-big-data.webp
electrical-installation-ai.webp
drone-inspection-ai.webp
```

## Sanitized case studies

For client or institution projects, publish only redacted one-pagers under:

```text
assets/case-studies/
```

Avoid original proposal decks, internal URLs, customer data, exact datasets, private diagrams, logos or identifiable field imagery unless there is explicit permission.
